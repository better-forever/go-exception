# GoException

#### 介绍
集合了golang API开发中常用的异常码和异常提示信息。

#### 软件架构
包含2个文件：
1.  exception.go：异常码和异常提示信息
2.  types.go：公共类

#### 安装教程

#### 使用说明

1.  生成一个新的异常使用New方法；
2.  1开头为系统异常，2开头为数据库异常，3开头为网络异常，4开头为业务异常；5开头为其他异常。
